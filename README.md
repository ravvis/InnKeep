# InnKeep

## Project setup
```
git clone git@github.com:singh-ravi-siso/InnKeep.git
cd InnKeep
npm install
```

### Compiles and hot-reloads for development

#### Server
```
npm run dev
```

#### Client
```
npm run build-dev --watch
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
